## MailMergeProject-Python


### Project Description:
- Using names found within a Txt file, to generate invitation letters for each person
- Combining logic and methods with a for-loop


### Purpose of the Project:

Gain an understanding of:
- How to read and write to external files using Python
- How to use new methods including .readlines(), .strip(), and .replace()
- using Absolute and Relative file paths
